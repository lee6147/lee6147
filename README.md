### loc_bot - 원격 개발 자동화
```
Telegram 메시지
    ↓
loc_bot (Python)
    ↓
Claude Code CLI
    ↓
코드 작성 / 수정 / 리뷰
    ↓
실시간 결과 스트리밍
    ↓
Telegram으로 응답
```

어디서든 Telegram으로 Claude Code를 제어하여
프로젝트 코드를 작성하고 리뷰받는 시스템을 직접 구축했습니다.

</td>
<td width="50%">

### Obsidian HTML Viewer - 지식 시각화
```
JSX/TSX 소스
    ↓
Claude Code + v4.4 프롬프트
    ↓
Obsidian 호환 단일 HTML 변환
    ↓
Custom Frames 자동 등록
    ↓
Obsidian에서 인터랙티브 가이드 열람
```

Claude Code를 활용하여 React 기반 인터랙티브 가이드를
