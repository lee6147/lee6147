# 자기소개 페이지

전자공학부 4학년 자기소개 포트폴리오 페이지입니다.

## 🚀 실행 방법

### 방법 1️⃣ - 가장 간단함 (추천)
1. `intro.html` 파일을 **더블클릭** 합니다
2. 또는 `intro.html` 파일을 마우스 우클릭 → "열기 방식" → **브라우저 선택**

### 방법 2️⃣ - 드래그
- `intro.html` 파일을 **Chrome, Edge, Firefox 등의 브라우저 아이콘**으로 드래그

### 방법 3️⃣ - 파일 탐색기에서
- 폴더를 열어서 `intro.html` → 우클릭 → "기본 앱으로 열기" → 웹 브라우저 선택

### 방법 4️⃣ - 명령어로 실행
```bash
# Windows PowerShell에서
start intro.html

# 또는 Git Bash에서
explorer intro.html
```

## 📝 파일 구성
- `intro.html` - 자기소개 페이지 (메인 파일)
- `README.md` - 설명서 (현재 파일)

## 💻 커스터마이징 방법
`intro.html` 파일을 텍스트 편집기(VS Code, 메모장 등)로 열어서:

1. **이메일 주소 수정** - 약 220번째 줄 근처
   ```html
   <a href="mailto:your@email.com" class="contact-link">📧 이메일</a>
   ```
   `your@email.com`을 본인 이메일로 변경

2. **기술 스택 수정** - 약 180번째 줄 근처
   ```html
   <span class="skill-tag">C/C++</span>
   ```
   원하는 기술로 추가/수정

3. **내용 수정** - 약 190번째 줄부터 원하는 섹션 수정
